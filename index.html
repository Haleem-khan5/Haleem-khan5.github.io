Hangman Game

current | HTML CSS NodeJS REACT MongoDB

Hangman Game is an engaging multiplayer web application where two players can participate, one as the word proposer and the other as the Hangman who guesses the word. The game allows only ten tries, and every correct guess reveals a part of the Hangman. After seven correct tries, the Hangman is complete, and you win; otherwise, the Word proposer wins. The game supports multiple games simultaneously, providing an immersive experience for players to enjoy with their friends or family members.

TeenPatti (Cards Game)

Apr 2023 HTML CSS NodeJS REACT MongoDB

Teen Patti is a popular card game played with four players, where each player plays on its own, meaning there is no coordination. The game is played with a deck of cards, and each player is dealt three face-down cards and three face-up cards. The objective of the game is to have the best three-card hand, with various combinations ranking higher than others. Teen Patti has become a favorite among card game enthusiasts worldwide due to its simple yet exciting gameplay mechanics. TeenPatti is sure to provide hours of entertainment for everyone who loves a good challenge.
